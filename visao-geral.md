# Visão Geral do Projeto: LootRadar (Tracker de Jogos Gratuitos)

## 1. O Problema
Jogadores frequentemente perdem a oportunidade de resgatar jogos 100% gratuitos (ofertas por tempo limitado na Epic Games, Steam, GOG) ou códigos de *loot* por não acompanharem as diversas lojas e fóruns diariamente. 

## 2. A Solução (A Aplicação)
O **LootRadar** é uma aplicação web comunitária que centraliza e exibe todos os jogos e *loots* que estão disponíveis de graça no momento. O sistema permite que os usuários vejam as ofertas ativas, filtrem por plataforma (PC, PlayStation, Xbox, Mobile) e marquem quais ofertas eles já resgataram, criando uma biblioteca pessoal de "Ganhos".

## 3. Público-Alvo
Comunidade gamer em geral, desde jogadores casuais que buscam novos jogos sem custo, até jogadores assíduos que colecionam resgates e códigos promocionais.

## 4. Casos de Uso Principais (User Stories)
- Como usuário, quero ver uma lista atualizada de jogos gratuitos na página inicial para saber o que posso resgatar hoje.
- Como usuário, quero poder filtrar as ofertas por plataforma (ex: apenas PC) para ver apenas o que me interessa.
- Como usuário, quero clicar em uma oferta e ser redirecionado para o link oficial de resgate.
- Como usuário, quero criar uma conta simples para salvar as ofertas que eu já resgatei, evitando ver a mesma oferta várias vezes.

## 5. Escopo da Disciplina
Este projeto cumpre os requisitos acadêmicos ao:
1. Resolver um problema da comunidade (acesso fácil a jogos gratuitos).
2. Consumir ativamente uma API Pública (GamerPower API).
3. Implementar a tríade clássica de desenvolvimento web: Front-end, Back-end e Banco de Dados.
